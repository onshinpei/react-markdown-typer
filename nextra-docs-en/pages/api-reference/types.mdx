## IntervalType

Typing interval type:

```typescript
type IntervalType = number | {
  max: number;      // Maximum interval (milliseconds)
  min: number;      // Minimum interval (milliseconds)
  curve?: 'ease' | 'ease-in' | 'ease-out' | 'ease-in-out' | 'linear' | 'step-start' | 'step-end';
  curveFn?: (x: number) => number;  // Custom curve function
}
```

## ITypedChar

Typed character callback data type:

```typescript
interface ITypedChar {
  currentIndex: number;    // Current character index
  currentChar: string;     // Current character
  prevStr: string;         // Previous string
  currentStr: string;      // Current full string
  percent: number;        // Progress percentage
}
```
