## Requirements

- React 16.8+ or higher
- Node.js 14.0+ or higher

## Using npm

```bash
npm install react-markdown-typer
```

## Using yarn

```bash
yarn add react-markdown-typer
```

## Using pnpm

```bash
pnpm add react-markdown-typer
```

## Verify Installation

After installation, you can verify it with:

```tsx
import MarkdownTyper from 'react-markdown-typer';

function App() {
  return (
    <MarkdownTyper interval={20}>
      # Installation Successful!
      
      If you can see this text, the installation was successful.
    </MarkdownTyper>
  );
}
```
