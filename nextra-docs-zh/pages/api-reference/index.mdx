react-markdown-typer 提供了两个主要组件：

## MarkdownTyper

声明式组件，通过 `children` 传入内容。

```tsx
<MarkdownTyper interval={20}>
  # Hello World
</MarkdownTyper>
```

## MarkdownTyperCMD

命令式组件，通过 `ref` 控制内容。

```tsx
const cmdRef = useRef<MarkdownTyperCMDRef>(null);
cmdRef.current?.push('# Hello World');

<MarkdownTyperCMD ref={cmdRef} interval={30} />
```
